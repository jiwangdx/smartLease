# 智慧公寓租赁平台 (SmartLease)

> 项目周期：2025.10 - 2025.12

## 项目简介

智慧公寓租赁平台是一个现代化的公寓租赁管理系统，包含**移动端用户平台**和**后台管理系统**两大核心模块。系统实现了用户登录认证、房源搜索、预约看房、租约管理等完整业务流程，为租客和房东提供高效便捷的租赁服务。

## 技术栈

| 技术分类 | 技术选型 |
|---------|---------|
| 后端框架 | SpringBoot |
| 数据库 | MySQL + Redis |
| 安全认证 | JWT |
| 对象存储 | MinIO |
| ORM框架 | MyBatisPlus |
| 并发处理 | ThreadPoolExecutor |

## 项目亮点与技术实现

### 1. 高性能缓存架构
- **热点房源二级缓存**：基于Redis设计二级缓存方案，有效提升系统QPS 40%+
- **用户状态隔离**：配合ThreadLocal实现用户状态线程级隔离，确保线程安全

### 2. 分布式安全认证
- **JWT双Token方案**：采用无状态鉴权，支持访问令牌与刷新令牌分离
- **Token黑名单机制**：结合Redis实现Token黑名单管理，实现安全可控的会话注销
- **自动续期机制**：优化用户体验，无需频繁重新登录

### 3. 异步任务与线程池优化
- **自研浏览记录异步处理组件**：提升用户浏览体验，降低同步处理延迟
- **内存去重算法**：避免重复记录，减少无效数据存储
- **CallerRunsPolicy拒绝策略**：保证高并发场景下的数据一致性

### 4. 对象存储安全升级
- **MinIO存储策略重构**：从公共读模式升级为私有访问模式
- **Presigned URL动态授权**：实现私有化可控访问，降低数据泄漏风险

### 5. 租约状态最终一致性保障
- **懒更新+定时巡检双机制**：减少数据库实时更新压力
- **乐观锁并发控制**：有效处理并发冲突，保证数据一致性

### 6. 全链路异常治理
- **Spring Boot AOP统一拦截**：构建全局异常处理体系
- **业务异常精准分类**：实现异常的精准定位与处理
- **链路追踪与结构化告警**：提升问题排查效率

## 快速开始

### 环境要求
- JDK 1.8+
- Node.js 16+
- MySQL 8.0+
- Redis 6.0+
- MinIO

### 后端启动

1. 初始化数据库：执行 `lease.sql` 脚本
2. 配置数据库、Redis、MinIO连接信息（修改各模块的application.yml）
3. 启动后台管理系统：`web-admin` 模块
4. 启动移动端服务：`web-app` 模块

### 前端启动

```bash
cd front
npm install
npm run dev
```
